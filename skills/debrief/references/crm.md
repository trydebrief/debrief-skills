# /debrief crm

Pull next steps, stage changes, and objections from a sales call into HubSpot.

## Apps
- **Granola** — pull the sales call transcript.
- **HubSpot** — match the deal/contact and write the note + field updates.

## Step 1: Identify the deal
Pull the transcript from Granola. Use HubSpot to match attendees to contacts and the associated deal. If multiple deals match, ask the user which.

## Step 2: Extract CRM-relevant signal
- **Stage change** (evidence from the transcript)
- **Next steps** (with owner and date)
- **Objections raised** and how they were handled
- **Competitors mentioned**
- **Budget/timeline/authority signals**

## Step 3: Draft the update
Show the proposed HubSpot note and any field changes (stage, close date, amount). Confirm before writing to HubSpot.

## Guidelines
- Never advance the stage without explicit verbal/textual confirmation.
- Preserve objection wording — sanitized summaries lose the signal.
- If the call contradicts existing CRM notes, flag it rather than overwriting.
