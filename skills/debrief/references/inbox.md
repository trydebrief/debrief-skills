# /debrief inbox

Priority inbox pass with drafted replies.

## Apps
- **Gmail** — search threads, read content, save drafts.

## Step 1: Categorize unread
Search Gmail for unread threads. Categorize:
- **Tier 1** — from key contacts or flagged urgent
- **Tier 2** — needs a reply but not urgent
- **Tier 3** — FYI, no action
- **Archive-worthy** — newsletters, notifications

## Step 2: Draft
For Tier 1 and Tier 2, use Gmail `create_draft` to save replies. Match the thread's tone and the user's voice.

## Step 3: Present
Summary: counts per tier, list of Tier 1 items with one-line context. Tell user where drafts are saved (Drafts label).

## Guidelines
- Never send automatically. Always save as a Gmail draft.
- Don't archive. Flag for archive, let the user confirm in a batch.
